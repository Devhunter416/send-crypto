## [0.2.0](https://github.com/renproject/send-crypto/compare/v0.1.14...v0.2.0) (2020-04-09)

The UTXO type has been changed from

```ts
interface SoChainUTXO {
    txid: string;
    value: number;
    output_no: number;
    confirmations: number;
    script_hex?: string;
}
```

to

```ts
interface UTXO {
    txHash: string;
    vOut: number;
    amount: number;
    confirmations: number;
    scriptPubKey?: string;
}
```
